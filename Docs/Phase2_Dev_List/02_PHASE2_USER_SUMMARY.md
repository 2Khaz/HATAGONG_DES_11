# Phase 2 현재 상태 요약 — 사용자용

## 한 줄 요약

Phase 2의 **기본 플레이 기능은 끝났고 실제 플레이 검수까지 통과**했습니다.  
지금 남은 것은 최종 리소스·UI·아이템·사운드·모바일 검증과 Phase 3 연결입니다.

---

## 지금 실제로 되는 것

- Phase 1이 끝나면 Phase 2로 전환됩니다.
- Phase 2에서 검은 영역을 좌클릭 드래그로 칠하면 회색 시멘트 바닥이 드러납니다.
- 좌클릭을 누른 채 보드 밖으로 나갔다가 다시 들어와도 도포가 이어집니다.
- 같은 곳을 다시 칠해도 점수와 진행도가 중복으로 오르지 않습니다.
- 내부 도포율 99%가 되면 클리어됩니다.
- 클리어 순간 점수가 확정됩니다.
- 남은 검은 자국은 약 0.4초 동안 자동으로 사라집니다.
- 화면이 완전히 회색으로 마감된 뒤 다음 Phase 전환을 요청합니다.
- 아직 Phase 3가 없기 때문에 마지막에는 안전하게 입력과 시간이 잠깁니다.

---

## 현재 확정 수치

- 전체 제한 시간: 90초
- Phase 2 목표 시간: 15~18초
- 어려운 난이도도 20초 이내 목표
- 판정 Grid: 128×128
- 시각 Mask: 1024×1024
- 클리어: 16,221/16,384셀
- 최대 점수: 1,450
- 브러시 크기:
  - 쉬움 8.5%
  - 보통 7.5%
  - 어려움 6.5%
- 자동 마감: 0.4초

- 진행도 게이지와 퍼센트
- 완료 광택·시멘트 퍼짐 연출
- 도포음·완료음·파티클·진동
